// 04 — SELECTED WORK

Things I've built

01 / 06

WordBot V2 — Word of the Day Bot

A Discord bot that runs a daily word challenge and grades sentence submissions through a multi-stage LLM pipeline. A heuristic prefilter blocks prompt-injection and score-manipulation attempts before the sentence is sanitized and graded, with MySQL-backed leaderboards, non-destructive season rollovers, scheduled automation, and moderation tools built in.

STACK — Python · Pycord · Groq · Gemini · MySQL
02 / 06

Tournament & Timer Management Bot

A live tournament management bot for Discord that runs concurrent, minute-aligned timers with deferred and pre-scheduled operations, a dual-reminder system, and guild whitelisting — backed by WAL-mode SQLite, interaction-safety middleware, and startup crash recovery for production reliability.

STACK — Python · Pycord · aiosqlite · asyncio
03 / 06

AI Discord Utilities & Automation

AI-powered Discord utilities including a single-file assistant bot with per-user memory, a two-tier prompt system, and admin controls — built with no external storage and no multi-file architecture — alongside broader automation tooling for moderation and scheduled tasks.

STACK — Python · Pycord · Groq
04 / 06

IoT Smart Waste Management System

An IoT system that monitors waste bin fill levels with Arduino and ultrasonic sensors, sending real-time alerts over GSM communication so collection can be scheduled by need rather than a fixed route.

STACK — Arduino · C++ · GSM · Sensors
05 / 06

Backend APIs & Automation Projects

A set of backend services and automation projects covering REST API design and database-backed workflows — built with Node.js, Express, and FastAPI to explore different approaches to the same backend problems.

STACK — Node.js · Express.js · FastAPI · REST APIs
06 / 06

Machine Learning Experiments

Applied ML experiments including a text classification competition tackled with a TF-IDF and calibrated LinearSVC baseline, then escalated to a multi-transformer ensemble with cross-validation, back-translation augmentation, pseudo-labeling, and optimized ensemble weighting.

STACK — Python · scikit-learn · Transformers
